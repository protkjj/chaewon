# 단어장

영어 단어를 추가하고, 낱말카드로 외울 수 있는 앱이야.

---

## 접속 방법

같은 Wi-Fi에 연결된 상태에서 Safari로 접속해:

```
http://172.30.1.21:8082
```

---

## 화면 구성

### 1. 홈 화면

앱을 열면 3개 버튼이 보여:

| 버튼 | 설명 |
|------|------|
| **단어장** | 알파벳별로 단어 보기 |
| **단어추가** | 하나씩 또는 대량으로 추가 |
| **낱말카드** | 전체 단어 학습 |

---

### 2. 단어장 (알파벳 그리드)

A~Z 알파벳 버튼이 나와. 각 버튼에 그 알파벳의 단어 수가 적혀 있어.

- 맨 앞에 노란 **★ 버튼**: 즐겨찾기한 단어만 모아보기
- 알파벳을 누르면 그 글자로 시작하는 단어 목록이 나와
- 오른쪽 위 휴지통 버튼: 삭제한 단어 확인/복원
- 오른쪽 위 검색 버튼: 전체 검색

---

### 3. 단어 목록 (알파벳별)

예를 들어 A를 누르면 A로 시작하는 단어들이 쭉 나와.

상단 고정 버튼:
- **낱말카드**: 이 알파벳 단어만 카드로 학습
- **+ 추가**: 단어 추가 화면으로 이동
- **뜻 숨기기**: 한글 뜻을 블러 처리 (영어만 보면서 자기 테스트 가능)

#### 단어 조작법

| 동작 | 방법 |
|------|------|
| 별표 (즐겨찾기) | 단어 영역 **두 번 터치** 또는 왼쪽 **☆ 버튼** |
| 수정/삭제 | 오른쪽 **⋯ 버튼** 탭 |

#### 뜻 숨기기

- **뜻 숨기기** 버튼을 누르면 모든 한글 뜻이 블러 처리됨
- 영어만 보면서 뜻을 떠올려볼 수 있어
- 특정 단어의 뜻만 보고 싶으면 **블러된 뜻 부분 탭** → 그 단어만 공개
- **뜻 보기** 버튼을 누르면 전체 공개

#### 단어 수정 모달 (⋯ 버튼)

수정 모달에서 할 수 있는 것:
- 영어 단어, 한국어 뜻 수정
- 즐겨찾기 토글
- 삭제 (한 번 더 확인 후 삭제, "삭제되었습니다" 표시)

---

### 4. 단어추가

두 가지 입력 방식이 있어:

#### 하나씩 입력 (기본)

1. 영어 단어 입력 → **다음** 버튼 또는 Enter
2. 뜻 입력 → **추가** 버튼 또는 Enter
3. 자동으로 다음 영어 입력 준비
4. 추가된 단어가 아래에 바로 표시됨

- 영어 입력칸에 한글을 쓰면 경고가 뜨고 넘어가지 않아
- 영어 단어는 자동으로 **소문자로 통일** 저장

#### 대량 입력

탭을 눌러 전환. **단어 — 뜻** 형식으로 여러 줄 한꺼번에 입력/붙여넣기.

줄바꿈 없이 붙여 써도 자동으로 구분됨:
```
apple — 사과 book — 책 accommodate — 수용하다, 숙박시키다
```

아래 형식 다 인식돼:

| 형식 | 예시 |
|------|------|
| em dash | `apple — 사과` |
| 하이픈 (공백 있음) | `apple - 사과` |
| 하이픈 (공백 없음) | `apple-사과` |
| 공백만 | `apple 사과` |

여러 뜻은 **콤마(,)** 로 구분:
```
accommodate — 수용하다, 숙박시키다, 부응하다
```

#### PDF 업로드

대량 입력 탭에서 **PDF** 버튼을 누르면 PDF 파일을 선택할 수 있어.
- PDF에서 텍스트를 자동 추출해서 입력칸에 채워줌
- 추출 결과를 확인/수정한 뒤 저장하면 됨

#### 중복 처리

이미 있는 단어를 다시 추가하면:

- 새로 만들지 않고 **기존 단어에 병합**됨
- 뜻이 새로 추가된 게 있으면 합쳐짐
  - 기존: `가, 나` + 새로 입력: `가, 나, 다` → `가, 나, 다`
- 기존 뜻이 더 많으면 줄어들지 않음
  - 기존: `가, 나` + 새로 입력: `가` → `가, 나` 그대로
- 영어 단어 옆에 주황색으로 **x2**, **x3** 같이 추가된 횟수가 표시됨

---

### 5. 검색

검색 버튼을 누르면 검색 화면이 나와.

- **영어**로 검색: `isomorphism` 입력하면 해당 단어 찾기
- **한국어**로 검색: `동형사상` 입력하면 뜻에 포함된 단어 찾기
- 타이핑하는 즉시 결과가 나옴
- 매칭된 부분은 **노란색**으로 하이라이트
- 검색 결과에서도 별표 토글, ⋯ 수정/삭제 가능

---

### 6. 낱말카드 (학습 모드)

퀴즈렛처럼 카드를 넘기면서 단어를 외울 수 있어.

| 동작 | 방법 |
|------|------|
| 카드 뒤집기 | 카드 **탭** |
| 별표 (즐겨찾기) | 카드 **두 번 탭** 또는 카드 안 **☆ 버튼** |
| 다음 카드 | **왼쪽으로 스와이프** 또는 ▶ 버튼 |
| 이전 카드 | **오른쪽으로 스와이프** 또는 ◀ 버튼 |
| 셔플 (순서 섞기) | 가운데 셔플 버튼 |
| 나가기 | 왼쪽 위 x 버튼 |

- 카드 앞면: 영어 단어 + 별표 버튼
- 카드 뒷면: 한국어 뜻 (여러 개면 번호 매겨서 표시)

학습 범위:
- **홈 → 낱말카드**: 전체 단어 학습
- **알파벳 → A → 낱말카드**: A 단어만 학습
- **★ → 낱말카드**: 즐겨찾기 단어만 학습

---

### 7. 휴지통

삭제한 단어가 여기에 보관돼. 알파벳 그리드에서 휴지통 버튼으로 접근.

- **복원** 버튼: 삭제한 단어를 다시 단어장으로 되돌림
- **x 버튼**: 완전히 삭제 (복구 불가)
- **전체 삭제**: 휴지통 비우기 (한 번 더 확인)

---

## 데이터 저장

- 모든 데이터는 **기기 안에** 저장돼 (인터넷 필요 없음)
- 앱을 껐다 켜도 단어가 그대로 남아 있어
- 삭제한 단어는 휴지통에서 복원 가능
- 단, Safari에서 **방문 기록 삭제**하면 데이터도 날아가니 주의!
- 나중에 iCloud/Firebase 연동으로 기기 간 동기화 예정
