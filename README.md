## 기기 간 동기화

아이폰/아이패드에서 같은 단어장을 쓰려면:

1. 홈 화면 상단 **"Google 로그인으로 기기 동기화"** 버튼 탭
2. Google 계정으로 로그인
3. **같은 Google 계정**으로 다른 기기에서도 로그인

- 한번 로그인하면 계속 유지됨
- 단어 추가/수정/삭제가 자동으로 동기화
- 로그인 안 해도 앱 사용은 가능 (기기 안에만 저장)

## 화면 구성

### 1. 홈 화면

| 버튼 | 설명 |
|------|------|
| **단어장** | 알파벳별로 단어 보기 |
| **단어추가** | 하나씩 또는 대량으로 추가 |
| **낱말카드** | 전체 단어 학습 |

---

### 2. 단어장 (알파벳 그리드)

A~Z 알파벳 버튼이 나와. 각 버튼에 그 알파벳의 단어 수가 적혀 있어.

- 맨 앞 노란 **★ 버튼**: 즐겨찾기한 단어만 모아보기
- 오른쪽 위 휴지통 버튼: 삭제한 단어 확인/복원
- 오른쪽 위 검색 버튼: 전체 검색

---

### 3. 단어 목록 (알파벳별)

상단 고정 버튼:
- **낱말카드**: 이 알파벳 단어만 카드로 학습
- **+ 추가**: 단어 추가 화면으로 이동
- **뜻 숨기기**: 한글 뜻을 블러 처리 (영어만 보면서 자기 테스트)

#### 단어 조작법

| 동작 | 방법 |
|------|------|
| 별표 (즐겨찾기) | 단어 영역 **두 번 터치** 또는 왼쪽 **☆ 버튼** |
| 수정/삭제 | 오른쪽 **⋯ 버튼** 탭 |

#### 뜻 숨기기

- **뜻 숨기기** 버튼을 누르면 모든 한글 뜻이 블러 처리됨
- 특정 단어의 뜻만 보고 싶으면 **블러된 뜻 부분 탭**
- **뜻 보기** 버튼을 누르면 전체 공개

#### 단어 수정 모달 (⋯ 버튼)

- 영어 단어, 한국어 뜻 수정
- 즐겨찾기 토글
- 삭제 (한 번 더 확인 후 삭제)

---

### 4. 단어추가

두 가지 입력 방식:

#### 하나씩 입력 (기본)

1. 영어 단어 입력 → **다음** 버튼 또는 Enter
2. 뜻 입력 → **추가** 버튼 또는 Enter
3. 자동으로 다음 영어 입력 준비
4. 추가된 단어가 아래에 바로 표시됨
5. 영어 입력칸에 한글 쓰면 경고

#### 대량 입력

탭을 눌러 전환. **단어 — 뜻** 형식으로 여러 줄 입력/붙여넣기.

줄바꿈 없이 붙여 써도 자동으로 구분됨:
```
apple — 사과 book — 책 accommodate — 수용하다, 숙박시키다
```

아래 형식 다 인식돼:
- `apple — 사과`
- `apple - 사과`
- `apple-사과`
- `apple 사과`

여러 뜻은 콤마(,)로 구분. 입력하면 아래에 파싱 미리보기가 뜸.

텍스트 기반 PDF 업로드도 가능 (PDF 버튼).

#### 중복 처리

- 이미 있는 단어 → 기존 단어에 뜻 병합
- 기존 뜻이 더 많으면 줄어들지 않음
- 추가 횟수가 주황색으로 표시 (x2, x3)
- 영어 단어는 자동 소문자 통일

---

### 5. 검색

- 영어/한국어 모두 검색 가능
- 실시간 결과 + 노란색 하이라이트
- 검색 결과에서도 별표, ⋯ 수정/삭제 가능

### 6. 낱말카드 (학습 모드)

| 동작 | 방법 |
|------|------|
| 카드 뒤집기 | 카드 **탭** |
| 별표 | 카드 **두 번 탭** 또는 카드 안 **☆ 버튼** |
| 다음 카드 | **왼쪽으로 스와이프** 또는 ▶ 버튼 |
| 이전 카드 | **오른쪽으로 스와이프** 또는 ◀ 버튼 |
| 셔플 | 가운데 셔플 버튼 |
| 나가기 | 왼쪽 위 x 버튼 |

#### 알아요 / 몰라요

- 카드 아래에 **알아요** / **몰라요** 버튼
- 상단에 현재 아는 단어 수 / 모르는 단어 수 표시
- 처음에는 전부 모르는 단어 (예: ✗ 93 / ✓ 0)
- 알아요 누르면 아는 단어로 이동

#### 학습 완료

모든 카드를 다 넘기면:
- **모르는 단어가 있으면** → "몰라요 N개만 다시 학습" (랜덤 재배치)
- **다 맞으면** → "모든 단어를 다 알아요!" + 처음부터 다시 / 돌아가기

#### 학습 범위

- **홈 → 낱말카드**: 전체 단어
- **알파벳 → A → 낱말카드**: A 단어만
- **★ → 낱말카드**: 즐겨찾기만

---

### 7. 휴지통

알파벳 그리드에서 휴지통 버튼으로 접근.

- **복원** 버튼: 삭제한 단어를 다시 단어장으로 되돌림
- **x 버튼**: 완전히 삭제 (복구 불가)
- **전체 삭제**: 휴지통 비우기

---

## 데이터 저장

- 모든 데이터는 **기기 안에** 저장 (오프라인에서도 사용 가능)
- Google 로그인하면 **Firebase에 자동 동기화**
- 같은 Google 계정 = 같은 단어장 (기기 상관없이)
- 중복 단어는 앱 시작 시 자동 제거
- Safari에서 **방문 기록 삭제**하면 로컬 데이터 + 로그인이 날아가니 주의
